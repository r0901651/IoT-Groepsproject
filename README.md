# IoT-Groepsproject

# Pinout Wiring

## RFID-RC522

| Pin  | Color  | Connection |
|------|--------|------------|
| VCC  | Red    | 5V         |
| GND  | Blue   | GND        |
| MISO | Black  | GPIO9      |
| MOSI | White  | GPIO10     |
| SCK  | Yellow | GPIO11     |
| SDA  | Green  | GPIO8      |

## HX711

| Pin  | Color  | Connection |
|------|--------|------------|
| VCC  | Red    | 5V         |
| GND  | Blue   | GND        |
| DT   | Black  | GPIO6      |
| SCK  | White  | GPIO5      |

## 28BYJ Driver

| Pin     | Color  | Connection |
|---------|--------|------------|
| 5-12V+  | Red    | 5V         |
| 5-12V-  | Blue   | GND        |
| IN1     | Yellow | GPIO17     |
| IN2     | Green  | GPIO27     |
| IN3     | Purple | GPIO22     |
| IN4     | Black  | GPIO26     |
